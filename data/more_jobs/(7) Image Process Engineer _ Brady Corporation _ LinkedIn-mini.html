About the job

At Code - A Brady Company, we are looking for an exceptionally talented Image Processing Engineer to join our growing technology team. The Image Processing Engineer will participate in new product development as well as improving existing products. Responsibilities include:


  • Design, develop, and test image processing and pattern recognition algorithms
  • Implement in C/C++ and optimize for embedded platforms
  • Debug, adapt, and improve image processing algorithms to support customer applications
  • Enhance image processing development tools under MS Windows using MS Visual Studio
  • Develop and maintain software that meets quality system requirements of Code Corporation
  • Support product specification and documentation
  • Support porting and testing of algorithms to embedded platforms

Qualifications:


  • 2+ years of C/C++ experience related to the image processing algorithm
  • Windows Software development experience using MS Visual Studio
  • BS/BA in EE, CS, Math or equivalent required. MS preferred
  • Experience in image processing, signal detection, and estimation, and pattern recognition
  • Strong problem-solving skills
  • Ability to work in a team environment
  • Must be a self-starter and detail-oriented
  • Experience in Linux and/or embedded systems desirable


Our hybrid work schedule combines on-premise and remote models and creates a unique (and more flexible) environment - one where team members alternate time in the office with a bit of remote work. You will have to live or relocate to the location.
