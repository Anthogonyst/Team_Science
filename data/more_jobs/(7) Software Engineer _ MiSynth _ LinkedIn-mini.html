About the job

MiSynth is a music software plugin startup that is empowering musicians, songwriters, and producers to create music by using the sounds they hear or imagine in their heads. MiSynth will make it easier for anyone to start producing music with unique sounds designed from your imagination and generated in seconds with our revolutionary technology. MiSynth is a diving board for your creativity where your imagination is the only limit.


The ideal candidate will be responsible for developing high-quality applications such as building and testing data acquisition methods. They will also be responsible for designing and implementing testable and scalable code.

  • Familiar with machine learning, Python and Javascript preferred
  • Modifying software to fix errors, adapt it to various hardware brain-computer interfaces (BCIs), improve its performance and upgrade interfaces.
  • Directing system testing and validation procedures
  • Directing software programming and documentation development
  • Designing and developing software systems using scientific analysis and mathematical models to predict and measure outcomes and design conesquences
  • Analyze and maintain existing software applications
  • Design highly scalable, testable code
  • Discover and fix programming bugs


Qualifications

  • Bachelor's degree or equivalent experience in Computer Science or related field
  • Development experience with programming languages
  • SQL database or relational database skills
  • Has an interest in music technology and neuroscience
